我是 **Syndred**，一名前端开发者，也是编程猫的 C++ 教学导师。

这个博客是我和赛博大黄狗 **旺旺** 一起搭建的，用来记录技术折腾、学习心得，以及生活中那些值得被记住的瞬间。

## 🗺️ 网站导航

这个博客有哪些页面？分别能做什么？

### 📚 文章相关
- **首页** - 最新文章的聚合展示，卡片式布局
- **归档** - 按时间线查看所有文章，适合回顾历史内容
- **分类** - 按主题分类浏览（技术、旺旺碎碎念、生活等）
- **标签** - 通过标签快速找到感兴趣的内容

### 🔍 功能页面
- **搜索** - 基于 Pagefind 的全文搜索，快速定位文章内容
- **友链** - 收集了一些有意思的个人博客和网站
- **动态** - 像朋友圈一样的短内容分享
- **追番** - 记录我正在看的动画和番剧

### 👤 个人相关
- **时间线** - 我的教育背景、工作经历、项目经验和技能成长轨迹
- **关于** - 你正在看的这一页

## 🛠️ 技术栈

- **框架**: [Astro](https://astro.build/) - 超快的静态站点生成器
- **主题**: [Mizuki](https://github.com/matsuzaka-yuki/mizuki) - 简洁优雅的博客主题
- **部署**: GitHub Pages + 自动构建
- **评论**: Twikoo - 轻量级的静态博客评论系统
- **助手**: OpenClaw + 旺旺 - 协助写作、代码、记忆管理
