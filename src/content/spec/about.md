## 🗺️ 网站导航

这个博客有哪些页面？分别能做什么？

### 📚 文章相关
- **首页** - 最新文章的聚合展示，技术or分享博客
- **归档** - 按时间线查看所有文章，适合回顾历史内容
- **分类** - 按主题分类浏览（技术、旺旺碎碎念、生活等）
- **标签** - 通过标签快速找到感兴趣的内容

### 🔍 功能相关
- **搜索** - 基于 Pagefind 的全文搜索，快速定位文章内容
- **链接** - 一些项目和平台地址
- **友链** - 一些不错的网站
- **相册** - 记录一些镜头📷
- **常用软件** - 目前比较常用的软件，MB表示手机端，PC表示电脑端

### 👤 个人相关
- **日记** - 偶尔更新
- **项目展示** - 过往写的一些东西，很多半成品😅😅
- **技能展示** - 目前掌握的技能
- **时间线** - 我的教育背景、工作经历、项目经验和技能成长轨迹
- **关于** - 你正在看的这一页

## 🛠️ 技术栈

- **框架**: [Astro](https://astro.build/) - 超快的静态站点生成器
- **主题**: [Mizuki](https://github.com/matsuzaka-yuki/mizuki) - 简洁优雅的博客主题，一眼心动
- **部署**: GitHub Pages + vercel自动构建
- **评论**: Twikoo - 轻量级的静态博客评论系统
- **助手**: OpenClaw - 协助写作、代码、记忆管理
