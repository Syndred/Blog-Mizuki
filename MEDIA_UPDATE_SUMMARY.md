## 下一步操作

### 1. 准备图片文件

你需要准备两张占位图片：

```
public/assets/books/chongfan.jpg      # 书籍封面占位图
public/assets/movies/Interstellar.webp # 电影海报占位图
```

### 2. 图片规格建议

- **尺寸**：400-600px 宽，2:3 比例（如 400x600）
- **格式**：jpg, png, webp
- **大小**：< 200KB

### 3. 获取图片方式

**方法1：使用现有图片**
- 如果你已经有《重返狼群》和《星际穿越》的图片，直接使用

**方法2：从豆瓣下载**
- 重返狼群：https://book.douban.com/subject/25862578/
- 星际穿越：https://movie.douban.com/subject/1889243/

**方法3：使用占位图片**
- 访问 https://via.placeholder.com/400x600.jpg?text=Book
- 访问 https://via.placeholder.com/400x600.jpg?text=Movie

### 4. 启动测试

```bash
# 确保图片文件存在
ls public/assets/books/chongfan.jpg
ls public/assets/movies/Interstellar.webp

# 启动开发服务器
npm run dev

# 访问页面
http://localhost:4321/media/
```
